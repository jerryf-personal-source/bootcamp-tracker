# ROADMAP — GitHub Bootcamp (Week 1 Intensive)

**Start:** Monday, March 2, 2026  
**Pace:** ~2–3 hours per weekday

## Day 1 — Local Git mastery (March 2)
**Outcome:** You control working tree ↔ staging ↔ commits, and can recover from mistakes locally.

**Drills**
- 25 micro-commits
- Use `git add -p` at least once
- Use `git restore` to undo a file change
- Amend a commit once
- Run git diff before every commit

**Break & recover**
- Delete a file; restore it
- Make a “bad” commit; fix it (amend or revert)

**Deliverable (app)**
- v0.1.0: Minimal window opens + basic layout loads from plan data

## Day 2 — Branching as AI containment (March 3)
**Outcome:** Every AI/agent experiment happens on a branch and merges through PR.

**Drills**
- 10 branches + PRs
- 3 merge conflicts resolved

**Deliverable (app)**
- v0.2.0: Checkboxes + save progress to `data.json`

## Day 3 — Damage control mastery (March 4)
**Outcome:** You can confidently undo, revert, reset, and recover via reflog.

**Drills**
- `git revert` used once
- `git reset --soft` used once (local only)
- `git reflog` recovery used once

**Deliverable (app)**
- v0.3.0: Daily confidence score + reflection notes

## Day 4 — GitHub governance (March 5)
**Outcome:** GitHub becomes your control center: Issues, PR discipline, protected main.

**Drills**
- Branch protection enabled
- PR template in place
- 10 Issues created/closed via PR (“Closes #…”)

**Deliverable (app)**
- v0.4.0: Completion % + streak indicator

## Day 5 — CI + Releases (March 6)
**Outcome:** You ship with confidence: CI checks + tags + release notes.

**Drills**
- GitHub Action runs on push + PR
- 3 tags/releases

**Deliverable (app)**
- v1.0.0: Week 1 tracker polished + release notes
